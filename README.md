# conversao-bases

Este projeto tem como objetivo criar um conversor de bases em C.

Após compilado, é exibido para o usuário as opções de conversão disponíveis.
Após selecionar a opção correspondente a conversão, é solicitado o número da base de origem.<br>
**Obs:** Algumas conversões solicitam também a base destino.


**Conversões disponíveis:**<br>
Decimal para binário<br>
Decimal para octal<br>
Decimal para outra base (necessário digitar a base destino)<br>
Binario para decimal<br>
Binario para octal<br>
Binario para outra base (necessário digitar a base destino)


Requisitos:
- Compilador c instalado em seu ambiente
